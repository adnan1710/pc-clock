# FluentFocus Clock

Minimal Electron + Vite + React clock with Fluent/Acrylic styling.

Run locally (install deps first):

```powershell
npm install
npm run electron:dev
```

Build for Windows (after `npm run build`):

```powershell
npm run build
npm run electron:build
```
